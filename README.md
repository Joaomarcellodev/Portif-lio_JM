## 📖 Sobre

O **Portfólio Profissional e Pessoal** é um projeto desenvolvido com **HTML**, **CSS** e **JavaScript**, pensado para apresentar minha trajetória, habilidades, projetos e formas de contato de forma clara, interativa e responsiva. 🧑‍💻✨  

### 🚀 Aprendizados
- Estruturação semântica com **HTML5**.
- Estilização moderna com **CSS3**.
- Interatividade e manipulação de elementos com **JavaScript**.
- Criação de um layout responsivo para diferentes dispositivos.
- Organização e apresentação de informações de forma profissional.
